# diff2xlsx

Convert diff to xlsx.

## Description

## Usage
Standard.

```bash
git --no-pager diff | diff2xlsx write -o diff.xlsx
```

See help.

```bash
diff2xlsx
diff2xlsx write -h
```

## Install

To install, use `go get`:

```bash
$ go get -d github.com/jiro4989/diff2xlsx
```
